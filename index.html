Project 2
JavaScript Pixelated Image Animation

Developed an interactive pixel-based animation in JavaScript to simulate a rainfall-like visual effect over an uploaded image. The project dynamically generates and manipulates approximately 5,000 individual pixel elements, creating the illusion of motion through randomized velocity, size variation, and speed adjustments. The animation logic was implemented using nested for-loops and JavaScript’s Math.random() function to introduce natural, non-uniform movement patterns and enhance realism. This project demonstrates core programming concepts such as iteration, randomness, and real-time DOM-based animation. Built using HTML5, CSS3, and vanilla JavaScript, with a focus on creative visual effects and algorithmic control of graphical elements.
